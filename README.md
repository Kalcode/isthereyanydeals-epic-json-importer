# IsThereAnyDeal Epic Games Importer

A web app to import your Epic Games library into [IsThereAnyDeal](https://isthereanydeal.com/) collections.

## Features

- Paste a JSON list of Epic Games
- Match games against the ITAD database
- Import to your ITAD collection

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Build for production
bun build

# Deploy to Cloudflare Workers
bun deploy
```

## Tech Stack

- [Astro](https://astro.build/) - Web framework
- [Solid-JS](https://www.solidjs.com/) - Reactive UI
- [Cloudflare Workers](https://workers.cloudflare.com/) - Edge deployment
